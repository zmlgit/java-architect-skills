# Spring Refactor

Expert Spring Boot refactoring with pattern detection and best practices.

## When to Use

Use this skill when you need to:
- Refactor large or complex Spring code
- Apply clean code principles
- Eliminate code smells
- Improve code maintainability

## Features

- Anti-pattern detection
- 20+ refactoring patterns
- Spring-specific best practices
- Interactive refactoring

## Tools

### `analyze`
Analyze code for refactoring opportunities.

### `apply`
Apply refactoring patterns interactively.
