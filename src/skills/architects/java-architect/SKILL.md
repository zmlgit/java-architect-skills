# Java Architect

Enterprise Java architecture review for Spring applications.

## When to Use

Use this skill when you need to:
- Review Java/Spring project architecture
- Assess SOLID principle adherence
- Identify design patterns and anti-patterns
- Verify clean architecture principles
- Evaluate domain-driven design implementation

## Features

- Layer-by-layer analysis
- SOLID principle scoring
- Design pattern detection
- Clean architecture verification
- DDD assessment

## Tools

### `review`
Comprehensive architecture review with customizable scope.

### `check-principles`
Focused SOLID and design principle check.
